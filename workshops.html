Python

MatLab with Python Hands-On Workshop

Join us for a hands on workshop from our MathWorks team on using MATLAB with Python! MathWorks is looking to create a connection point for conversations about the landscape of computational languages - and the broader impact that open source and commercial software have in academia and industry today. In this hands-on workshop, you’ll learn how to use Python with MATLAB using practical examples. Specifically, you’ll learn how to:

  • Call Python libraries from MATLAB
  • Call user-defined Python commands, scripts, and modules
  • Manage and convert data
  • Call MATLAB from Python
  • Package MATLAB algorithms to be called from Python

Date: October 4th @ 4:00PM - 5:00PM
Location: ISEB 1010
